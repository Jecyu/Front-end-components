## 技术栈和编译工具

原生`ES6+`、`scss`，配合`rollup`打包工具，编译成`UMD`规范。

## 如何运行

`git clone git@github.com:Jecyu/front-end-components.git`

`yarn`

开发`yarn dev`

生产`yarn build`

打包全部文件 `yarn buildAll`
